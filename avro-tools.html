Abstract

Apache avro-tools dependency allows to operate with serialization/deserialization of records in AVRO format, this dependency is used under the hood in streaming systems like Apache Kafka for records processing, also as a binary format in other framework. But it can be applied as a standalone executable to explore files and records in AVRO format. Here is the list of basic operations with avro-tools. Schemas and binary files from the Avro schema evolution previous post are used in this article.

      compile  Generates Java code for the given schema.

     fromjson  Reads JSON records and writes an Avro data file.
     fromtext  Imports a text file into an avro data file.
      getmeta  Prints out the metadata of an Avro data file.
    getschema  Prints out schema of an Avro data file.

 idl2schemata  Extract JSON schemata of the types from an Avro IDL file
       induce  Induce schema/protocol from Java class/interface via reflection.

      recodec  Alters the codec of a data file.

      rpcsend  Sends a single RPC message.
